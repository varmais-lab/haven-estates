Google Sheets Setup
Step-by-step connection guide
Step 1 — Create Your Google Sheet

Go to sheets.google.com and create a new spreadsheet. Add these column headers in Row 1:

Timestamp | Name | Mobile | Email | Profession | Budget | Industry | City | AccessType | Interest | PropertyId | PropertyTitle
Step 2 — Create Apps Script

In your sheet: Extensions → Apps Script. Paste this code:

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet()
    .getSheetByName('Registrations');
  const data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.timestamp, data.name, data.mobile, data.email,
    data.profession, data.budget, data.industry, data.city,
    data.accessType, data.interest, data.propertyId, data.propertyTitle
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({status:'success'}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet()
    .getSheetByName('Registrations');
  const data = sheet.getDataRange().getValues();
  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}
Step 3 — Deploy as Web App

Click Deploy → New Deployment. Set type to Web App. Execute as Me. Access: Anyone. Copy the deployment URL and paste it into config.json → googleSheets.scriptUrl.

Step 4 — Test & Go Live

Go to Settings tab above, enter your Script URL, and click Test Connection. Once confirmed, all registration form submissions will appear in your Google Sheet within seconds.
